# Journey Journal

Journey Journal is an Android mobile application designed to assist users in planning, organizing, and managing their travel experiences efficiently. The application enables users to create trips, manage itineraries, track expenses, maintain checklists, and receive timely reminders for planned activities and tasks.

The application is developed using the Flutter framework and utilizes local storage with SQLite, allowing it to function entirely offline without the need for a backend server or internet connectivity.


## Target Platform

- Android

This application is developed and optimized exclusively for Android devices.


## Technology Stack

- Flutter (Mobile application framework)
- SQLite (Local database for persistent storage)
- Shared Preferences (Lightweight local data storage)
- GoRouter (Application routing and navigation)
- Flutter Local Notifications (Notification and reminder system)

## Application Features

- Trip creation and management
- Itinerary management for each trip
- Expense tracking per trip
- Checklist creation and management
- Notification and reminder system for:
  - Itinerary activities
  - Checklist items
- Offline-first functionality using local storage

## How to Run the Project

### Prerequisites

- Flutter SDK (version 3.0 or higher)
- Android Studio or Visual Studio Code
- Android Emulator or a physical Android device

---

### Setup Instructions

1. Clone the project repository:
```bash
git clone https://github.com/Chhun-Nika/JourneyJournal_app
```
2.	Navigate to the project directory
```bash
cd JourneyJournal_app
```
3. Install project dependencies:
```bash
flutter pub get
```
4. Run the application on an Android device:
```bash
flutter run
```
Ensure that an Android emulator is running or a physical Android device is properly connected before executing the application.
